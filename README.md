# papercert

Script that dumps a key and certificate to a PDF for printing.

## Installation

```shell
# For CentOS/RHEL/Fedora
git clone ${THIS_REPO}
sudo yum install -y dmtx-utils ImageMagick
```

## Usage

```shell
./papercert.sh key.pem cert.pem
```
